# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| v1.1.x | Yes |
| earlier | Best effort |

## Reporting a Vulnerability

セキュリティ上の問題は、公開 Issue ではなく非公開で報告してください。

公開の場に以下を投稿しないでください。

- API keys
- access tokens
- secret files
- 個人情報
- exploit detail

## Notes

このプロジェクトはローカル中心の開発支援ツールです。特に以下を重視します。

- 外部コマンド実行の安全性
- ファイル読み書きの安全性
- 意図しない危険操作の回避
