# Contributing to AI Loop Orchestrator

AI Loop Orchestrator へのコントリビュートを歓迎します。

## Development Setup

```bash
npm install
npm run compile
```

VS Code でこのリポジトリを開き、`F5` で Extension Development Host を起動してください。

## Guidelines

- 大きな変更の前に Issue で方針共有を推奨
- 無関係な変更を同じ PR に混ぜない
- README と実装の整合を保つ
- できれば再現手順やスクリーンショットを添付する

## Suggested Commit Prefixes

- `fix:`
- `feat:`
- `docs:`
- `refactor:`
